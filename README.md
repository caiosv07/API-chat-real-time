#  API Chat Real Time

Uma API de chat em tempo real construída com **Python**, **Django**, **Django REST Framework**, **PostgreSQL**, **Redis** e **WebSockets**, totalmente conteinerizada com **Docker Compose**.  


---

##  Tecnologias

- **Python**
- **Django**
- **Django REST Framework**
- **Django Channels**
- **Redis** (para WebSockets / Channel Layer)
- **PostgreSQL**
- **Docker & Docker Compose**

---

##  Instalação e Execução (Docker)

Certifique-se de ter **Docker** e **Docker Compose** instalados.

```bash
git clone https://github.com/caiosv07/API-chat-real-time.git
cd API-chat-real-time

docker-compose up --build
